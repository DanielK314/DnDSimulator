New BeastForms:
1.add Beast Entity to Archive
2.add to Beast Forms in Entity Class

New Other_Abilities: 
1.add Ability to Entity Class Init under special Abilites
2.Implement the feature in the code
3.Add Name to All_Abilities in GUI Controller Class
4.Change it in the GUI build_ability_page, just change the Number for the added Class feature

New Spells:
1. Add Spell Stats to the Spell Calss __init__
2. Add a Cast Function for the Spell
2.1 remember empowered Spells 
2.1 How to Handle Targtes
2.3 make spell/cantrip check
2.4 Add 1 to Cast Counter 
2.5 Change Concentration if necessary, Break Concentration
3. Add a Score Function for the Spell
3.1 must return a dmg equal score, target and spelllevel
4. Add Spell to Spell Name List in Entity Class
4.1 Add cast Counter in Entity 
5. If concentration Spell: check break concentration 
5.1 in the upper line 'lost concentraation'
5.2 Implement the break concentration in dead and unconscious function
5.3 remember to rest counter or variables of the spell at end of turn or long rest
6. The Spell should be listed automatically in GUI
6.1. Add to Spellrecap

New Token relation:
1. Create a Link Token, via the LinkToken class or via a subclass
2. Initiate one or more Link Token
3. Create a DockToken or a subclass of a DockToken
3.1 wenn Initiateing the DockToken, give a List of the Link Dokens.
3.2 The Link Tokens will be resolved if the Dock is, or vici verca if all links are resolved
4. You can add resolve conditions to the Tokens
5. You might add smt to the update function of the TM
